Что такое Ruby и Rails?

Ruby - динамический язык программирования, чтобы делать программистов счастливыми.

Автор - Юкихира Мацумото



Ruby on Rails - фреймворк (программный каркас) для профессиональной разработки web-приложений.

Автор - Давид Хейнемейер Ханссон

Несколько фактов о Ruby

  • Динамический язык. Динамические типы. Никакой компиляции;
  • Язык сверхвысокого уровня;
  • Воспитывает правильное отношение к программированию: Test Driven Development (TDD), Don’t Repeat Yourself (DRY);
  • Большое и отзывчивое сообщество;
  • Развитое метапрограммирование, идеален для создания DSL;
  • Подходит для освоения с нуля;
  • Более 142,385 готовых библиотек (gems) для разных задач.

А что же с Rails?

  • Идеально для построения web приложений любой сложности;
  • Любимец стартапов. Увеличивает скорость разработки на 30-40%;
  • Из коробки работает со всеми современными базами данных;
  • Основан на парадигме Model-View-Controller;
  • Принцип “convention over configuration”;
  • Используется в больших компаниях: Basecamp, Airbnb, Shopify, Github, Twitter, Hulu, Dribble, Groupon;
  • Прост в масштабировании
  • Удобный deploy: любимый capistrano используют и на Python и на PHP!

PHP


      function read_data($source)
      {
          if (method_exists($source, 'read')) {
              return $source->read();
          } elseif (is_string($source)) {
              $file = new SplFileObject($source, "r"));
              return $file->fread($file->getSize());
          }
          throw new InvalidArgumentException;
      }
      
      $filename = "foo.txt";
      read_data($filename); #=> считывает содержимое foo.txt 
                            # с помощью вызова SplFileObject->read();
      
      $input = new SplFileObject("foo.txt", "r");
      read_data($input); #=> считывает содержимое foo.txt  
                 # с помощью передачи в дескриптор файла

Ruby


      def read_data(source)
        return source.read if source.respond_to?(:read)
        return File.read(source.to_str) if source.respond_to?(:to_str)
        raise ArgumentError
      end
      
      filename = "foo.txt"
      read_data(filename) #=> считывает содержимое foo.txt 
            # с помощью вызова File.read() 
      
      input = File.open("foo.txt")
      read_data(input) #=> считывает содержимое foo.txt 
            # с помощью передачи в дескриптор файла
